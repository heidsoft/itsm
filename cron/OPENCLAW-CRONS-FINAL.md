## ⚠️ 注意事项

### 通知管理
- **高频通知**（15 分钟 -4 小时）：构建失败、新 Issues
- **中频通知**（每天）：安全检查、质量报告
- **低频通知**（每周）：文档更新、项目周报

### 审查频率
- **feature 分支**: 每天审查
- **文档更新**: 每周审查
- **社区管理**: 每天查看

### 资源消耗
- **API 调用**: GitHub API 有速率限制（5000 次/小时）
- **计算资源**: 文档生成可能耗时
- **存储**: 历史报告会占用空间
