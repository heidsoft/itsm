# ROI 计算器

**版本**: 1.0  
**日期**: 2026-02-28  
**用途**: 帮助客户计算投资回报率

---

## 📊 计算器说明

本 ROI 计算器帮助客户量化 ITSM 平台的投资价值，包括：
- 成本节省
- 效率提升
- 风险降低
- 投资回报率

---

## 💰 成本计算

### 当前成本（不使用 ITSM）

#### 1. 人力成本

```
IT 团队人数：[输入] 人
平均年薪：[输入] 元/年
年人力成本 = 人数 × 平均年薪

示例：
IT 团队：50 人
平均年薪：¥200,000
年人力成本 = 50 × 200,000 = ¥10,000,000
```

#### 2. 培训成本

```
新人数量：[输入] 人/年
培训周期：[输入] 月
培训成本 = 新人数量 × 培训周期 × 月薪

示例：
新人：10 人/年
培训周期：3 个月
月薪：¥20,000
年培训成本 = 10 × 3 × 20,000 = ¥600,000
```

#### 3. 故障损失

```
年故障次数：[输入] 次
平均故障时长：[输入] 小时
每小时损失：[输入] 元
年故障损失 = 故障次数 × 平均时长 × 每小时损失

示例：
年故障：20 次
平均时长：4 小时
每小时损失：¥50,000
年故障损失 = 20 × 4 × 50,000 = ¥4,000,000
```

#### 4. 管理成本

```
管理人员：[输入] 人
管理工具：[输入] 元/年
年管理成本 = 管理人员年薪 + 管理工具费用

示例：
管理人员：5 人
年薪：¥200,000
管理工具：¥100,000
年管理成本 = 5 × 200,000 + 100,000 = ¥1,100,000
```

#### 5. 当前总成本

```
当前总成本 = 人力成本 + 培训成本 + 故障损失 + 管理成本

示例：
当前总成本 = 10,000,000 + 600,000 + 4,000,000 + 1,100,000 = ¥15,700,000/年
```

---

### 使用 ITSM 后成本

#### 1. 人力成本节省

```
效率提升：[输入]% (通常 30-50%)
节省人力成本 = 当前人力成本 × 效率提升%

示例：
效率提升：40%
节省人力成本 = 10,000,000 × 40% = ¥4,000,000
```

#### 2. 培训成本节省

```
培训周期缩短：[输入]% (通常 50-70%)
节省培训成本 = 当前培训成本 × 培训周期缩短%

示例：
培训周期缩短：67% (从 3 个月到 1 个月)
节省培训成本 = 600,000 × 67% = ¥400,000
```

#### 3. 故障损失减少

```
故障减少：[输入]% (通常 60-80%)
减少故障损失 = 当前故障损失 × 故障减少%

示例：
故障减少：75%
减少故障损失 = 4,000,000 × 75% = ¥3,000,000
```

#### 4. 管理成本节省

```
管理效率提升：[输入]% (通常 40-60%)
节省管理成本 = 当前管理成本 × 管理效率提升%

示例：
管理效率提升：55%
节省管理成本 = 1,100,000 × 55% = ¥605,000
```

#### 5. ITSM 投入成本

```
软件费用：[输入] 元/年
实施费用：[输入] 元（一次性）
培训费用：[输入] 元/年
维护费用：[输入] 元/年

年投入成本 = 软件费用 + 培训费用 + 维护费用

示例：
软件费用：¥500,000/年
实施费用：¥200,000（一次性）
培训费用：¥50,000/年
维护费用：¥50,000/年
年投入成本 = 500,000 + 50,000 + 50,000 = ¥600,000/年
```

#### 6. 使用 ITSM 后总成本

```
使用 ITSM 后总成本 = 当前总成本 - 节省成本 + ITSM 投入成本

节省成本 = 人力节省 + 培训节省 + 故障节省 + 管理节省
         = 4,000,000 + 400,000 + 3,000,000 + 605,000
         = ¥8,005,000

使用 ITSM 后总成本 = 15,700,000 - 8,005,000 + 600,000
                  = ¥8,295,000/年
```

## 📈 ROI 计算

### 投资回报率 (ROI)

```
ROI = (收益 - 投入) / 投入 × 100%

收益 = 节省成本 = ¥8,005,000
投入 = ITSM 投入成本 = ¥600,000

ROI = (8,005,000 - 600,000) / 600,000 × 100%
    = 7,405,000 / 600,000 × 100%
    = 1234%
```

### 投资回收期

```
投资回收期 = 投入成本 / 月收益

月收益 = 年收益 / 12 = 8,005,000 / 12 = ¥667,083/月

投资回收期 = 600,000 / 667,083 = 0.9 个月 ≈ 1 个月

（如果考虑一次性实施费用）
投资回收期 = (600,000 + 200,000) / 667,083 = 1.2 个月
```

### 3 年总收益

```
3 年总收益 = 年收益 × 3 - 一次性投入
          = 8,005,000 × 3 - 200,000
          = 24,015,000 - 200,000
          = ¥23,815,000
```

## 💡 使用示例

### 示例 1: 中型互联网企业

**企业背景**:
- 员工：500 人
- IT 团队：30 人
- 年故障：15 次

**当前成本**:
- 人力成本：30 × ¥250,000 = ¥7,500,000
- 培训成本：5 × 3 × ¥25,000 = ¥375,000
- 故障损失：15 × 3 × ¥100,000 = ¥4,500,000
- 管理成本：3 × ¥250,000 + ¥50,000 = ¥800,000
- **总成本**: ¥13,175,000/年

**使用 ITSM 后**:
- 人力节省：¥7,500,000 × 50% = ¥3,750,000
- 培训节省：¥375,000 × 67% = ¥250,000
- 故障节省：¥4,500,000 × 75% = ¥3,375,000
- 管理节省：¥800,000 × 55% = ¥440,000
- **总节省**: ¥7,815,000/年

**ITSM 投入**:
- 专业版：¥500,000/年
- 实施：¥150,000（一次性）
- 培训：¥30,000/年
- **年投入**: ¥530,000/年

**ROI 计算**:
- ROI = (7,815,000 - 530,000) / 530,000 × 100% = 1375%
- 回收期 = (530,000 + 150,000) / (7,815,000/12) = 1.04 个月
- 3 年收益 = 7,815,000 × 3 - 150,000 = ¥23,295,000

---

### 示例 2: 大型金融机构

**企业背景**:
- 员工：5000 人
- IT 团队：200 人
- 年故障：30 次

**当前成本**:
- 人力成本：200 × ¥300,000 = ¥60,000,000
- 培训成本：20 × 3 × ¥30,000 = ¥1,800,000
- 故障损失：30 × 2 × ¥500,000 = ¥30,000,000
- 管理成本：10 × ¥300,000 + ¥200,000 = ¥3,200,000
- **总成本**: ¥95,000,000/年

**使用 ITSM 后**:
- 人力节省：¥60,000,000 × 40% = ¥24,000,000
- 培训节省：¥1,800,000 × 67% = ¥1,200,000
- 故障节省：¥30,000,000 × 80% = ¥24,000,000
- 管理节省：¥3,200,000 × 55% = ¥1,760,000
- **总节省**: ¥50,960,000/年

**ITSM 投入**:
- 企业版：¥2,000,000/年
- 实施：¥500,000（一次性）
- 培训：¥100,000/年
- **年投入**: ¥2,100,000/年

**ROI 计算**:
- ROI = (50,960,000 - 2,100,000) / 2,100,000 × 100% = 2327%
- 回收期 = (2,100,000 + 500,000) / (50,960,000/12) = 0.61 个月
- 3 年收益 = 50,960,000 × 3 - 500,000 = ¥152,380,000
