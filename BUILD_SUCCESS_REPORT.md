
### 导入别名配置

在 `tsconfig.json` 中配置：

```json
{
  "paths": {
    "@/*": ["./src/*"],
    "@/components/*": ["./src/components/*"],
    "@/lib/*": ["./src/lib/*"],
    "@/types/*": ["./src/types/*"],
    "@/app/*": ["./src/app/*"]
  }
}
```
