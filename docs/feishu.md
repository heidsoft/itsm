# 飞书连接器配置和使用指南

## 概述
飞书连接器实现了 ITSM 系统与飞书的双向同步，支持以下功能：
1.  ITSM 工单自动同步到飞书任务
2. 飞书任务变更自动同步到 ITSM 工单
3. 飞书 OAuth 授权登录
4. 飞书消息卡片通知
5. 飞书事件回调处理

## 飞书开放平台配置
### 1. 创建企业自建应用
在 [飞书开放平台](https://open.feishu.cn/) 创建企业自建应用，获取以下信息：
- App ID
- App Secret
- Verification Token (用于事件回调验证)
- Encrypt Key (用于事件内容加密)

### 2. 配置权限
在应用的"权限管理"中开通以下权限：
| 权限名称 | 权限描述 | 是否必须 |
|---------|---------|---------|
| 任务:读写 | 用于同步工单到飞书任务 | 是 |
| 通讯录:只读 | 用于同步用户信息，映射 ITSM 用户与飞书用户 | 是 |
| 消息与群组:发送消息 | 用于发送工单通知卡片 | 是 |
| 事件订阅:接收事件 | 用于接收任务变更、消息回调等事件 | 是 |

### 3. 配置事件订阅
在应用的"事件订阅"中添加以下事件：
| 事件名称 | 事件类型 | 描述 |
|---------|---------|---------|
| 任务创建 | `task.created` | 飞书创建任务时触发，同步到 ITSM |
| 任务更新 | `task.updated` | 飞书任务更新时触发，同步到 ITSM |
| 任务删除 | `task.deleted` | 飞书任务删除时触发，同步状态到 ITSM |
| 接收消息 | `im.message.receive_v1` | 接收用户发送给机器人的消息 |
| 卡片回调 | `card.action.trigger` | 处理用户点击卡片按钮的事件 |

事件回调地址配置为：`https://your-itsm-domain.com/api/v1/feishu/webhook`

## ITSM 系统配置
### 1. 环境变量配置
在 `itsm-backend/.env` 中添加以下配置：
```env
# 飞书连接器配置
FEISHU_APP_ID=your_app_id
FEISHU_APP_SECRET=your_app_secret
FEISHU_VERIFICATION_TOKEN=your_verification_token
FEISHU_ENCRYPT_KEY=your_encrypt_key
# 飞书区域配置：cn（中国大陆）/intl（海外）
FEISHU_REGION=cn
```

### 2. 启用连接器
在系统设置 -> 连接器管理中启用飞书连接器，填写上述配置信息。

## 功能使用
### 1. 工单同步
- **自动同步**：ITSM 中创建或更新工单时，会自动同步到飞书任务
- **手动同步**：在工单详情页点击"同步到飞书"按钮，可手动触发同步
- **同步字段**：
  - 工单标题 -> 任务名称
  - 工单描述 -> 任务描述
  - 工单优先级 -> 任务优先级（低/中/高/紧急）
  - 工单状态 -> 任务状态（未开始/进行中/已完成/已取消）
  - 工单创建时间 -> 任务开始时间
  - 工单处理人 -> 任务负责人（需要用户映射配置）

### 2. 飞书任务同步到 ITSM
在飞书中创建或更新任务时，会自动同步到 ITSM 工单，默认创建人为系统管理员，可配置用户映射关系。

### 3. OAuth 授权
用户可以通过飞书 OAuth 授权登录 ITSM 系统，无需单独注册账号。

### 4. 消息通知
工单状态变更、分配、评论等事件会自动发送飞书卡片通知给相关人员。

## 用户映射配置
为了正确同步工单处理人到飞书任务负责人，需要配置 ITSM 用户与飞书用户的映射关系：
1. 在系统设置 -> 用户管理中，编辑用户信息
2. 在"飞书 Open ID"字段中填写用户的飞书 Open ID
3. 保存后即可自动映射处理人

## 常见问题
### Q: 同步失败，提示"应用权限不足"
A: 请检查飞书开放平台中应用是否开通了所有 required 的权限，并且权限已经过企业管理员审批。

### Q: 事件回调无响应
A: 请检查事件回调地址是否正确，服务器是否开放了公网访问，并且防火墙允许飞书 IP 段访问。

### Q: 任务状态不同步
A: 请检查事件订阅中是否添加了任务创建、更新、删除事件，并且事件状态为已启用。

### Q: 同步的任务负责人为空
A: 请检查用户映射是否配置正确，确保 ITSM 用户的飞书 Open ID 字段已填写。
