## 🔧 设置步骤

### 1. 注册 Formspree

1. 访问 https://formspree.io/
2. 注册账号（使用 heidsoft@qq.com）
3. 创建新表单
4. 获取表单 ID

### 2. 配置表单端点

**表单 action**:
```html
<form action="https://formspree.io/f/FORM_ID" method="POST">
```

**当前配置**:
```html
<form action="https://formspree.io/f/heidsoft" method="POST">
```

### 3. 验证邮箱

1. 登录 Formspree
2. 验证 heidsoft@qq.com 邮箱
3. 激活表单
