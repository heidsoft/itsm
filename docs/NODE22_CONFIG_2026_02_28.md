### 2. Package.json - Engine 要求

**文件**: `itsm-frontend/package.json`

```json
{
  "name": "itsm-prototype",
  "version": "0.1.0",
  "engines": {
    "node": ">=22.0.0",  # ✅ 要求 Node 22+
    "npm": ">=10.0.0"    # ✅ 要求 npm 10+

```
