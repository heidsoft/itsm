## 🎯 部署方式

### 1. Docker 部署

```bash
docker run -d \
  --name openclaw \
  -p 3000:3000 \
  -e API_KEY=your_api_key \
  openclaw/openclaw:latest
```

### 2. Kubernetes 部署

```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: openclaw
spec:
  replicas: 3
  template:
    spec:
      containers:
      - name: openclaw
        image: openclaw/openclaw:latest
        ports:
        - containerPort: 3000
```
