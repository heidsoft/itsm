# 通用UI组件测试用例

## 功能概述
测试通用UI组件（表单、按钮、表格、弹窗、导航等）的交互和样式。

## 测试用例

| 用例ID | 用例名称 | 前置条件 | 测试步骤 | 预期结果 |
|--------|----------|----------|----------|----------|

| UI-FORM-004 | 下拉选择 | 无 | 1. 展开下拉<br>2. 选择选项 | 选项选中 |
| UI-FORM-005 | 日期选择器 | 无 | 1. 打开日期选择<br>2. 选择日期 | 日期正确显示 |
| UI-FORM-006 | 文件上传 | 无 | 1. 选择文件<br>2. 上传 | 上传进度显示 |
| UI-FORM-007 | 多选框 | 无 | 1. 选择多个选项<br>2. 确认 | 多个选项选中 |
