## 1. 测试范围与目标

本文档覆盖ITSM系统工作流/BPMN模块的全部核心功能点，包括流程设计器、节点类型、流程部署、流程实例、用户任务、流程监控、自动化规则和流程权限等八大功能模块。

**测试目标**:
- 验证BPMN 2.0流程的正确解析与执行
- 确保流程实例生命周期的完整性
- 验证任务领取、完成任务、委派、转交等核心操作
- 确保网关条件判断的准确性
- 验证流程部署与版本管理
- 确保监控指标的准确性

```xml
<?xml version="1.0" encoding="UTF-8"?>
<definitions xmlns="http://www.omg.org/spec/BPMN/20100524/MODEL"
             id="Definitions_1"
             targetNamespace="http://bpmn.io/schema/bpmn">
```

```xml
    <userTask id="UserTask_Low" name="低优先级处理"/>
    <endEvent id="EndEvent_1"/>
    <sequenceFlow id="Flow_High" sourceRef="ExclusiveGateway_1" targetRef="UserTask_High">
      <conditionExpression xsi:type="bpmn:tFormalExpression" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">${amount > 1000}</conditionExpression>
    </sequenceFlow>
    <sequenceFlow id="Flow_Low" sourceRef="ExclusiveGateway_1" targetRef="UserTask_Low">
      <conditionExpression xsi:type="bpmn:tFormalExpression">${amount <= 1000}</conditionExpression>
    </sequenceFlow>
```
