## 测试用例状态定义

| 状态 | 说明 |
|------|------|
| 待评审 | 用例编写完成，待评审 |
| 已评审 | 用例已通过评审 |
| 已实现 | 对应测试代码已实现 |
| 已执行 | 用例已执行 |
| 阻塞 | 因依赖问题阻塞 |
| 通过 | 执行通过 |
| 失败 | 执行失败 |
| 跳过 | 因环境或需求变更跳过 |
