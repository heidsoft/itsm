| 属性 | 值 |
|------|-----|
| 用例ID | TC-KB-001 |

| 测试优先级 | P0 |
| 测试类型 | 功能测试 |

**前置条件:**
- 用户已登录系统
- 用户具有创建文章权限

**测试步骤:**
1. 点击"新建文章"按钮
2. 填写文章标题: "IT服务管理最佳实践"

6. 上传一张附件图片
7. 点击"保存草稿"按钮

**预期结果:**
- 文章创建成功
- 系统返回文章ID
- 文章状态为"草稿"
- 文章信息正确保存到数据库
