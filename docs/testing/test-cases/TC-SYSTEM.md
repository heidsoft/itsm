# 系统管理模块测试用例

## 功能概述
测试用户管理、角色管理、权限管理、部门管理、租户管理等系统管理功能。

| SYS-USR-004 | 删除用户 | 管理员 | 1. 选择用户<br>2. 点击删除<br>3. 确认 | 用户被删除 |
