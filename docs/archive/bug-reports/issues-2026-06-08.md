## 一、安全问题 (SEC)

### SEC-1: [已修复] 后端 API 代理缺少认证检查
- **文件**: `itsm-frontend/src/app/api/[...path]/route.ts`
- **问题**: 前端代理到后端时未验证 JWT token
- **严重性**: CRITICAL
- **修复**: 添加 JWT 验证，阻止敏感路径访问
- **状态**: ✅ 已修复

### SEC-2: [已修复] 硬编码密码
- **文件**: `itsm-backend/scripts/init_data/main.go`, `itsm-backend/pkg/seeder/seeder.go`
- **问题**: `admin123`, `itsm_password_2026` 等硬编码密码
- **严重性**: CRITICAL
- **修复**: 改用环境变量 `ADMIN_PASSWORD`
- **状态**: ✅ 已修复

### SEC-3: [已修复] Prometheus metrics 端点公开
- **文件**: `itsm-backend/router/router.go:203`
- **问题**: `/metrics` 端点无认证，可被互联网任意访问
- **严重性**: CRITICAL
- **修复**: 添加 JWT 认证中间件
- **状态**: ✅ 已修复

### SEC-4: [已修复] 租户隔离 - 全局搜索默认回退到 tenant_id=1
- **文件**: `controller/global_search_controller.go:52`
- **问题**: 缺少租户上下文时默认使用 tenant_id=1，跨租户数据泄露
- **严重性**: CRITICAL
- **修复**: 返回错误而非默认回退
- **状态**: ✅ 已修复

### SEC-5: [已修复] 租户隔离 - 知识库搜索默认回退
- **文件**: `handlers/knowledge/handler.go:274-278`
- **问题**: 同上，缺少 tenant_id 时默认回退到 1
- **严重性**: HIGH
- **修复**: 显式返回 400 错误
- **状态**: ✅ 已修复

### SEC-6: [已修复] end_user 角色可写 SLA 配置
- **文件**: `middleware/rbac.go:253`
