## 📋 优化概述

本次优化针对工作流模块的三个核心组件进行了功能增强和体验优化：

1. **审批组管理页面** - 添加成员管理功能
2. **审批节点配置面板** - 添加快捷操作和响应式优化
3. **工作流设计器** - 改进移动端适配

## 🎯 具体改进内容

### 1. 审批组管理页面 (admin/groups)

#### 新增功能

| 功能 | 描述 | 状态 |
|------|------|------|
| **成员管理** | 查看、添加、移除组成员 | ✅ |
| **批量添加** | 通过穿梭框批量选择用户 | ✅ |
| **成员预览** | 实时预览当前组成员列表 | ✅ |
| **Badge 展示** | 显示每个组的成员数量 | ✅ |

#### 技术实现

```typescript
// 新增 API 调用
- GroupAPI.getMembers(groupId)     // 获取组成员
- GroupAPI.addMember(groupId, userId)  // 添加成员
- GroupAPI.removeMember(groupId, userId) // 移除成员
```

#### 界面改进

- 表格新增"成员"列，显示成员数量 Badge
- 新增成员管理模态框，支持穿梭框选择用户
- 操作列新增"成员"快捷按钮

---

### 2. 审批节点配置面板 (WorkflowNodeInspector)

#### 新增功能

| 功能 | 描述 | 状态 |
|------|------|------|
| **快捷操作栏** | 常用配置一键设置 | ✅ |
| **配置状态 Badge** | 显示已配置的审批组数量 | ✅ |
| **操作提示** | 展示快捷操作提示 | ✅ |
| **刷新按钮 Tooltip** | 改善按钮可发现性 | ✅ |

#### 快捷操作

- 设为管理员 (assignee = admin)
- 普通优先级 (priority = 50)
- 24小时超时 (dueDate = PT24H)

#### 界面优化

- 用户任务区域新增快捷操作栏
- 候选组字段添加 Badge 数量显示
- 空状态添加操作提示

---

### 3. 工作流设计器 (WorkflowDesigner)

#### 新增功能

| 功能 | 描述 | 状态 |
|------|------|------|
| **响应式布局** | 适配移动端/平板/桌面 | ✅ |
| **移动端优化** | 调整间距和尺寸 | ✅ |
| **面板折叠** | 右侧属性面板响应式显示 | ✅ |

#### 布局改进

```css
/* 响应式断点 */
- 移动端 (< 768px): 垂直布局，属性面板全宽

```
