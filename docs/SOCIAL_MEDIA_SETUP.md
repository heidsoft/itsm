# 社交媒体 setup 指南

**目标**: 建立品牌影响力，获取精准客户

---

## 微信公众号

### 注册流程

1. 访问 https://mp.weixin.qq.com/
2. 注册账号（选择企业类型）
3. 填写企业信息
4. 等待审核（1-3 工作日）
5. 认证（¥300/年）

### 内容规划

**发布频率**: 每周 2-3 篇

**内容类型**:
- 产品更新（20%）
- 行业洞察（30%）
- 客户案例（20%）
- 技术教程（30%）

**目标**: 3 个月 10,000 粉丝

## 知乎

### 机构号 setup

1. 访问 https://www.zhihu.com/
2. 注册机构号
3. 填写企业信息
4. 认证（¥600/年）

### 内容策略

**回答方向**:
- ITSM 相关问题
- IT 运维管理问题
- 工单系统选择
- 数字化转型

**目标**: 3 个月 5,000 关注
