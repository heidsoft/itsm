# 菜单模块端到端回归测试清单

- 日期: 2025-12-07
- 目标: 逐一按系统菜单模块测试功能、交互、布局、A11y、权限/租户隔离、错误处理与性能基础
- 浏览器地址: <http://localhost:3000/>

## 全局与登录

- 登录/刷新令牌/未授权拦截；错误提示与焦点返回
- 路由守卫与租户切换（请求头 `Authorization`、`X-Tenant-ID/Code`）
- 全局 Toast 与 Loading/Empty/Error 模板统一
- 键盘导航与焦点陷阱（对话框/抽屉）

## 仪表盘 Dashboard

- 加载 KPI/趋势/分布/满意度/近期动态
- 筛选时间范围与图表联动；导出与卡片化
- 错误与空态显示一致

## 工单 Tickets

- 列表分页与筛选（状态/优先级/搜索/时间范围），URL 同步
- 详情查看与操作（编辑/删除），批量导出/删除
- 新建工单：分类/模板/标签与附件上传（API 层上传）
- 通知/评论/附件/评分/视图/模板/分类/标签/子任务/智能分配与规则
- 权限：创建/更新/删除/分配/导入导出；租户隔离
- A11y：搜索聚焦、表格导航、对话框焦点管理

## 事件 Incidents

- 列表分页与筛选（状态/优先级/来源/类型），统计一致性
- 详情更新与根因/影响评估、AI 分析
- 权限与租户隔离；错误与空态模板

## 问题 Problems

- 列表、详情、调查步骤 CRUD、根因报告与解决方案、摘要
- 审批/流程联动与审计

## 变更 Changes

- 列表/新建/详情；审批状态机一致性
- 工作流联动与日志

## 工作流 Workflow

- 设计器加载与部署；实例与任务操作；版本与自动化
- BPMN 表单与焦点管理；权限

## CMDB

- CI 列表/筛选/拓扑与关系图；影响评估
- 大数据渲染与虚拟滚动；错误与空态模板

## 服务目录 Service Catalog

- 列表与统计；请求流程（审批/完成/评分/收藏）
- 表单校验与撤回/重试

## 知识库 Knowledge Base

- 文章 CRUD 与分类/标签；版本与协作；关联工单与推荐

## SLA

- 定义 CRUD；合规检查；违规列表与状态更新；预警规则与历史

## 报表 Reports

- 过滤与时间范围；导出 CSV/PNG；保存为仪表盘卡片与分享链接

## 通知 Notifications

- 列表与过滤；已读/未读；用户偏好设置

## 管理台 Admin

- 用户/角色/权限/租户/部门/团队/项目/应用/系统配置
- CRUD 与成员管理；树形结构展示；权限角色关联；租户切换流程

## 模板 Templates/标签 Tags

- 模板编辑器与列表；标签 CRUD 与绑定

## Profile/我的请求

- 个人信息与偏好；我的请求列表与筛选

## 验收项（一致性）

- API 合同一致（`code/message/data`）；错误码与提示准确
- 列表页使用统一筛选/分页与 Loading/Empty/Error 模板
- A11y：键盘导航、焦点可见、ARIA 语义与 `aria-live` 公告
- 权限与租户隔离（越权拦截与头部校验）
- 性能基础：分页、缓存去重、防抖；大列表滚动流畅
