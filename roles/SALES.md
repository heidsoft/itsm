# 销售总监 (Sales Director)

**角色代码**: SALES-001  
**创建时间**: 2026-02-28  
**汇报对象**: CEO/创始人  
**下属团队**: 销售团队、客户成功团队

---

## 📋 角色概述

负责 ITSM 产品的市场推广、销售策略制定和客户获取，实现产品商业化目标和收入增长。

---

## 🎯 核心职责

### 1. 销售策略制定

- 制定产品定价策略和套餐方案
- 设计销售渠道和合作伙伴计划
- 制定季度/年度销售目标和计划
- 分析市场竞争格局和定价策略

### 2. 市场开拓

- 识别目标客户群体和市场细分
- 制定市场推广和获客策略
- 建立销售漏斗和转化流程
- 组织产品演示和试用活动

### 3. 客户管理

- 建立客户关系管理体系 (CRM)
- 制定客户成功和留存策略
- 管理大客户和战略合作伙伴
- 收集客户反馈并推动产品改进

### 4. 销售团队建设

- 招聘和培训销售人员
- 制定销售激励和考核机制
- 建立销售流程和标准话术
- 监控销售绩效和转化率

## 📞 销售话术

### 开场白

**场景**: 首次电话沟通

```
"您好，我是 ITSM 平台的 [姓名]。我们注意到贵公司在 IT 服务管理方面可能有一些挑战，
我们帮助过 [类似公司] 提升了 50% 的 IT 运维效率。

想了解一下，贵公司目前是如何管理 IT 工单和事件的呢？"
```

### 需求挖掘

**问题清单**:

1. "目前 IT 团队有多少人？每天处理多少工单？"
2. "现有的工单管理流程是怎样的？有哪些痛点？"
3. "是否有 SLA 管理？达标率如何？"
4. "有没有考虑过引入专业的 ITSM 系统？"
5. "预算范围大概是多少？"

### 价值主张

**核心卖点**:

1. **提升效率**: "帮助客户平均提升 50% 的 IT 运维效率"
2. **降低成本**: "减少 30% 的人力成本投入"
3. **合规审计**: "满足 ITIL 最佳实践和审计要求"
4. **AI 赋能**: "智能分类、自动派单、根因分析"
5. **快速上线**: "标准化部署，2 周即可上线使用"

### 异议处理

**异议 1**: "价格太贵了"

**回应**:
```
"理解您的考虑。其实从长期来看，我们的系统可以帮助您：
- 减少 30% 的人力成本
- 降低 50% 的系统故障时间
- 提升 40% 的客户满意度

算下来，投资回报率通常在 6 个月内就能收回。
而且我们可以根据您的实际情况调整套餐方案。"
```

**异议 2**: "我们现在用 Excel/钉钉就够了"

**回应**:
```
"Excel/钉钉确实在初期够用，但当工单量增长后会出现：
- 无法追踪处理进度
- 没有 SLA 时效管理
- 数据统计分析困难
- 知识无法沉淀

我们的系统专门解决这些问题，而且可以和钉钉/企业微信集成，
不影响现有使用习惯。"
```

**异议 3**: "需要考虑一下"

**回应**:
```
"完全理解，这是一个重要决策。

为了帮助您更好地评估，我可以：
1. 安排一个 30 天的免费试用
2. 提供同行业的客户案例
3. 安排一次技术答疑会议

您看哪个对您更有帮助？"
```
