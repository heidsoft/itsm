# 财务总监 (CFO/Finance Director)

**角色代码**: FINANCE-001  
**创建时间**: 2026-02-28  
**汇报对象**: CEO/创始人  
**下属团队**: 财务团队、会计团队

---

## 📋 角色概述

负责公司财务管理、资金运营、财务规划和风险控制，确保公司财务健康，支持业务决策和战略发展。

---

## 🎯 核心职责

### 1. 财务规划与分析 (FP&A)

- 制定年度财务预算和经营计划
- 进行财务预测和滚动预测
- 分析财务指标和经营绩效
- 为战略决策提供财务支持

### 2. 会计核算

- 建立和完善会计核算体系
- 监督日常财务核算工作
- 编制财务报表和合并报表
- 确保会计信息准确性和及时性

### 3. 资金管理

- 管理公司现金流和资金运作
- 制定资金计划和融资策略
- 维护银行关系和融资渠道
- 优化资金使用效率

### 4. 税务管理

- 制定税务筹划方案
- 监督税务申报和缴纳
- 处理税务稽查和争议
- 优化税务结构和税负

### 5. 内部控制

- 建立和完善内控制度
- 监督财务流程和审批权限
- 进行内部审计和风险评估
- 确保合规经营

### 6. 投资者关系

- 管理投资者关系和沟通
- 准备融资材料和路演
- 参与投资谈判和尽职调查
- 维护股东关系

## 💰 定价与收入确认

### SaaS 收入确认原则

#### 1. 订阅收入

**确认方式**: 按服务期间分期确认

**示例**:
```
客户购买 1 年期专业版：¥120,000
- 签约当月确认：¥10,000
- 后续 11 个月每月确认：¥10,000
```

**会计处理**:
```
签约收款时:
  借：银行存款 120,000
  贷：预收账款 120,000

每月确认收入时:
  借：预收账款 10,000
  贷：主营业务收入 10,000
```

#### 2. 项目实施收入

**确认方式**: 按完工百分比法确认

**示例**:
```
项目实施合同：¥500,000
- 项目启动 (30%): ¥150,000
- 中期验收 (40%): ¥200,000
- 终验 (30%): ¥150,000
```

#### 3. 定制开发收入

**确认方式**: 按里程碑确认

**示例**:
```
定制开发合同：¥1,000,000
- 需求确认 (20%): ¥200,000
- 系统设计 (30%): ¥300,000
- 开发完成 (30%): ¥300,000
- 上线验收 (20%): ¥200,000
```

## 📋 财务制度

### 1. 费用报销制度

#### 报销流程

```
员工提交 → 部门审批 → 财务审核 → 出纳付款 → 会计入账
  ↓          ↓          ↓          ↓          ↓
1 个工作日  2 个工作日  2 个工作日  1 个工作日  1 个工作日
```

#### 报销标准

| 费用类型 | 标准 | 说明 |
|---------|------|------|
| 差旅费 | 实报实销 | 需提供发票和行程单 |
| 住宿费 | ¥500/天 | 一线城市¥800/天 |
| 餐饮费 | ¥200/天 | 需提供发票 |
| 交通费 | 实报实销 | 优先公共交通 |
| 招待费 | ¥500/次 | 需提前申请 |

### 2. 采购管理制度

#### 采购审批权限

| 金额范围 | 审批人 |
|---------|--------|
| <¥10,000 | 部门经理 |
| ¥10,000 - ¥50,000 | 财务总监 |
| ¥50,000 - ¥200,000 | CEO |
| >¥200,000 | 董事会 |

#### 采购流程

```
需求申请 → 比价询价 → 合同审批 → 验收入库 → 付款申请
  ↓          ↓          ↓          ↓          ↓
1 个工作日  3 个工作日  2 个工作日  1 个工作日  2 个工作日
```

### 3. 合同管理制度

#### 合同审批流程

```
业务发起 → 法务审核 → 财务审核 → 领导审批 → 合同盖章
  ↓          ↓          ↓          ↓          ↓
1 个工作日  2 个工作日  2 个工作日  2 个工作日  1 个工作日
```

#### 合同归档

- 原件由财务部统一保管
- 扫描件上传合同管理系统
- 建立合同台账和档案
- 定期清理到期合同
