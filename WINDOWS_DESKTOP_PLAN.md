# ITSM Windows 桌面版规划

## 🎯 产品定位

基于 Web 版 ITSM 系统，打造原生 Windows 桌面应用，提供更佳的用户体验！

## 💡 技术选型

### 推荐方案：Electron + React

#### 优势
- ✅ **技术复用** - 直接使用现有 React 代码
- ✅ **开发效率** - 一套代码，多端运行
- ✅ **生态成熟** - Electron 生态完善
- ✅ **跨平台** - 可同时发布 Windows/Mac/Linux

#### 技术栈
```
框架：Electron 28+
前端：React 19 + TypeScript 5
UI: Ant Design 6
打包：electron-builder
自动更新：electron-updater
```

### 备选方案：Tauri

#### 优势
- ✅ **体积更小** - 比 Electron 小 10 倍
- ✅ **性能更好** - 使用系统 WebView
- ✅ **内存更省** - 内存占用更低

#### 劣势
- ❌ 需要 Rust 开发能力
- ❌ 生态相对较新
- ❌ 学习曲线陡峭

## 🏗️ 架构设计

### 项目结构
```
itsm-desktop/
├── src/
│   ├── main/          # Electron 主进程
│   │   ├── main.ts    # 主进程入口
│   │   ├── preload.ts # 预加载脚本
│   │   └── ...
│   ├── renderer/      # Electron 渲染进程
│   │   └── (复用 itsm-frontend 代码)
│   └── shared/        # 共享代码
├── package.json
├── electron-builder.yml
└── ...
```

### 技术架构
```
┌─────────────────┐
│  Renderer 进程  │  ← React 应用 (复用 Web 版)
│  (Chromium)     │
└────────┬────────┘
         │ IPC
┌────────┴────────┐
│   Main 进程     │  ← Node.js (系统 API)
│   (Node.js)     │
└─────────────────┘
```
