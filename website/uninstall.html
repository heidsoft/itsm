🗑️ 卸载指南

完整、干净地卸载 OpenClaw

卸载前须知

⚠️ 重要提示

卸载前请备份重要配置文件，特别是包含 API Key 的 .env 文件。卸载后数据将无法恢复。

建议备份的文件：

  • ~/.openclaw/workspace/openclaw.json - 主配置文件
  • ~/.openclaw/workspace/.env - 环境变量（含敏感信息）
  • ~/.openclaw/logs/ - 日志文件（如需审计）

Linux / macOS 卸载

步骤 1: 停止服务

终端命令
openclaw stop

步骤 2: 卸载 OpenClaw

终端命令
npm uninstall -g openclaw

步骤 3: 删除配置目录

终端命令
rm -rf ~/.openclaw

步骤 4: 删除开机自启（如果设置了）

Linux
sudo systemctl disable openclaw sudo rm /etc/systemd/system/openclaw.service sudo systemctl daemon-reload
macOS
launchctl unload ~/Library/LaunchAgents/openclaw.plist rm ~/Library/LaunchAgents/openclaw.plist

步骤 5: 验证卸载

终端命令
openclaw --version # 应该显示：command not found ls -la ~/.openclaw # 应该显示：No such file or directory

Windows 卸载

步骤 1: 停止服务

PowerShell
openclaw stop

步骤 2: 卸载 OpenClaw

PowerShell
npm uninstall -g openclaw

步骤 3: 删除配置目录

PowerShell
Remove-Item -Recurse -Force $env:USERPROFILE\.openclaw

步骤 4: 删除开机自启（如果设置了）

PowerShell
Remove-Item "$env:APPDATA\Microsoft\Windows\Start Menu\Programs\Startup\OpenClaw.lnk"

步骤 5: 验证卸载

PowerShell
openclaw --version # 应该显示：The term 'openclaw' is not recognized Test-Path $env:USERPROFILE\.openclaw # 应该显示：False

卸载完成
