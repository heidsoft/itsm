📖 钉钉机器人创建指南

  1. 访问 open.dingtalk.com
  2. 创建机器人应用
  3. 获取 AppKey 和 AppSecret

📖 飞书机器人创建指南

  1. 访问 open.feishu.cn
  2. 创建机器人应用
  3. 获取 App ID 和 App Secret

📖 企业微信机器人创建指南

  1. 访问 work.weixin.qq.com
  2. 创建机器人应用
  3. 获取 BotID 和 Secret

📖 Telegram Bot 创建指南

  1. 在 Telegram 搜索 @BotFather
  2. 发送 /newbot 创建机器人
  3. 获取 Bot Token

📖 Discord Bot 创建指南

  1. 访问 discord.com/developers
  2. 创建 Application → Bot
  3. 获取 Bot Token 和 Application ID

📖 Slack App 创建指南

  1. 访问 api.slack.com
  2. 创建新应用 → 添加 Bot 权限
  3. 获取 Bot Token
