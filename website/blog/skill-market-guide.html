📖 什么是 Skill？

Skill 是 OpenClaw 的能力扩展模块，就像手机的 App 一样。通过安装不同的 Skill，你可以让 OpenClaw 具备各种专业技能。

  • 🔍 搜索技能 - 网络搜索、知识库检索
  • 📁 文件技能 - 文件操作、数据导出
  • 💻 执行技能 - 命令执行、脚本运行
  • 📊 分析技能 - 数据分析、报表生成

📥 安装 Skill

方法 1: 通过配置生成器

  1. 访问配置生成器页面
  2. 选择需要的平台和技能
  3. 生成配置并下载
  4. 运行安装脚本

方法 2: 手动安装

# 1. 进入技能目录 cd ~/.openclaw/workspace/skills/ # 2. 克隆技能仓库 git clone https://github.com/openclaw/skill-dingtalk.git # 3. 安装依赖 cd skill-dingtalk npm install # 4. 配置技能 cp config.example.json config.json # 编辑 config.json 填写配置 # 5. 启用技能 openclaw skill enable dingtalk

⚙️ 管理 Skill

查看已安装的 Skill

openclaw skill list # 输出示例： ✅ dingtalk - v1.2.0 - 已启用 ✅ web-search - v2.0.1 - 已启用 ⏸️ file-ops - v1.0.0 - 已禁用

启用/禁用 Skill

# 启用技能 openclaw skill enable file-ops # 禁用技能 openclaw skill disable dingtalk

更新 Skill

# 更新单个技能 openclaw skill update dingtalk # 更新所有技能 openclaw skill update-all

🔧 配置 Skill

查看 Skill 配置

openclaw skill config dingtalk

修改 Skill 配置

# 编辑配置文件 vim ~/.openclaw/workspace/skills/dingtalk/config.json # 配置示例： { "appKey": "dingxxxxxxxxxx", "appSecret": "your_secret", "enabled": true, "logLevel": "info" }

🧪 测试 Skill

运行技能测试

openclaw skill test dingtalk # 输出示例： Running tests for dingtalk... ✅ Connection test passed ✅ Message send test passed ✅ Webhook test passed All tests passed!
