- **布局嵌套**：通过 `layout.tsx` 实现了高效的布局复用。
- **代码组织**：`src/lib` (核心逻辑) 与 `src/components` (UI组件) 职责分明。

### 2. API 集成 (严禁模拟数据)
在 [http-client.ts](file:///Users/heidsoft/Downloads/research/itsm/itsm-frontend/src/lib/api/http-client.ts) 中实现了生产级 HTTP 客户端：
- **规范转换**：自动处理后端 `snake_case` 与前端 `camelCase` 的互转，无缝对接 Go 后端。
- **安全认证**：实现了 Token 自动刷新机制 (401拦截) 和租户 ID/Code 自动注入。

- **组件质量**：[Button.tsx](file:///Users/heidsoft/Downloads/research/itsm/itsm-frontend/src/components/ui/Button.tsx) 等基础组件封装完善，支持多变体、加载状态及移动端触摸优化。
